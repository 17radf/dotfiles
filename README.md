install 
=======

```shell script

# first setup the ssh key for github
# make a new directory
git clone git@github.com:17radf/wyvern.git ~/.dotfiles

# install 
cd ~/.dotfiles
./install

```

setup details
=============

- arch linux
- kitty w/ zsh
- i3-gaps, polybar, rofi, etc
- nvim
